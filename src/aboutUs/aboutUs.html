superMeal

superMeal website is designed to allow users to search for delicious meals and view random meal suggestions. It fetches data from the TheMealDB API to retrieve meal information and display it to the user. The main purpose for creating this website was only for hobby, joy and want to give some help for everybody. I'm deeply hoping that everyone will be enjoy browsing through the website.

Features üî©

Navigation

The website includes a navigation bar with links to the home page and an "About Us" page.

Search Bar

Users can enter a search query in the search bar to find meals matching their query. When the search form is submitted, an API request is made to retrieve meals with the specified name.

Random Meals

The main section of the website displays a grid of random meals. These meals are fetched from the meal API using an API endpoint. Users can click the "More Meals..." button to load additional random meals.

Search Results

When a user submits a search query, the search results section is displayed. It shows the meals that match the search query. If no meals are found, appropriate error messages are shown.

Meal Details

By double-clicking on a meal in either the random meals or search results sections, a dialog box is opened, showing detailed information about the selected meal. The dialog box includes the meal's name, image, category, area, tags, ingredients, measures, and instructions.

Favourite Meals

If a user click on a heart icon, the meal will store in localStorage API in browser. By doing so, users can save their favourite meals and it won't delete even the browser. Users can view and delete their favourite meals as they want.

Usages üìù

Open the website in a web browser.

Use the navigation links to explore different pages.

Enter a meal name in the search bar and click the search button to search for specific meals.

Browse the random meals displayed on the main page.

Double-click on a meal to view its detailed information in a dialog box.

Click the "More Meals..." button to load additional random meals.

Enjoy searching and exploring delicious meals!
